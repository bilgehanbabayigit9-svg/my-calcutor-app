# C++ Calculator App

A simple console-based calculator written in C++.

## Features
- Supports addition (+)
- Supports subtraction (-)
- Supports multiplication (*)
- Supports division (/)

## How to Use
Enter values in this order:

1. First number
2. Second number
3. Operator (+, -, *, /)

Example:

10
20
+
Result: 30

This project was created to practice basic C++ programming concepts.
